# RayCaster
A Raycaster written in C using the SDL library for window creating and drawing logic

## Requirements

- C compiler (GCC or Clang)
- CMake (version 3.10+)
- SDL2 development libraries

## Build & Run

```bash
git clone https://github.com/jackwdoddsy/RayCaster.git
cd RayCaster
mkdir build
cd build
cmake ..
cmake --build .
./raycaster
```
